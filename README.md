# Citizen Signal Network — Lesotho

**Prototype v0.1** | UNDP Innovation Challenge Submission

A citizen engagement, feedback, and public project monitoring 
digital platform built for Lesotho's governance context.

## Live Prototype
👉 https://mosala-qekisi.github.io/citizen-signal-network/

## What this prototype demonstrates
- Citizen report submission (web + USSD channel design)
- Public project monitoring map with district-level progress
- "You Said – We Did" accountability dashboard
- Open consultation portal
- Offline-first PWA architecture (mobile-first, low-bandwidth)
- Bilingual interface (English / Sesotho toggle)

## Full production stack (as proposed)
- Backend: Django (Python) + PostgreSQL + PostGIS
- Frontend: Progressive Web App (React)
- Containerisation: Docker + Nginx
- USSD/SMS: RapidPro
- Licence: European Union Public Licence (EUPL)
